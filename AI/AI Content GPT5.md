Project Ideas
	- taken from GPT5 while working on a portfolio template

* Network Monitoring Dashboard
	* real-time visualization of network health using WebSockets and D3.js

* Product Management Kanban
	* react-based Kanban board with drag-and-drop, offline support, and dark theme

* Technical Writing Site
	* static site with Hugo, hosting API docs and guides with syntax highlighting

* Cloud Cost Optimizer
	* scheduled analyzer that flags idle resources and rightsizes instances across clouds.

* Incident Runbook Generator 
	* CLI that turns past incident data into Markdown runbooks with checklists.

* Home Lab Kubernetes 
	* HA k3s cluster on mini-PCs with GitOps, metrics, and SSO via Keycloak.

* API Rate Limiter
	* Redis-backed token bucket middleware with per-tenant quotas and burst control

* Accessibility Audit Tool
	* Automated a11y check via Axe + Lighthouse with CI gates and trend charts. 

* Static Site Theme
	* Dark-mode Hugo theme with semantic color tokens and zero JS by default.

* Portfolio Generator CLI
	* Scaffolds a portfolio from JSON, emits responsive images, and Open Graph tags

* Passwordless Auth Demo
	* WebAuthn + Passkeys example with server-side verification and device sync

* Data Viz Gallery
	* D3/Observable gallery of charts with theming and accessible annotations. 

* Personal Finacne Tracker
	* PWA for budgeting with offline storage, encryption at rest, and importer.
